# Pointe Games

Kids Ministry Game Suite by The Pointe Church, Cocoa Beach FL.

## Games
- **Phrase Frenzy** — Wheel of Fortune-style word game
- **Word Drop** — Bible verse memory game
- **Tic Tac Toe** — Trivia team battle
- **The Grid** — Bible Jeopardy
- **Scoreboard** — Team score tracker
- **Countdown** — Game timer
- **No Whammies** — Press Your Luck
- **Clue Climb** — Pyramid word game
- **Game Hub** — All games in one app
